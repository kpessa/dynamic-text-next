# Welcome to Dynamic Text Editor Storybook

This is the component library for the Dynamic Text Editor application, built with **Next.js 15**, **React**, and **Material UI**.

## 🏗️ Architecture

We follow **Feature-Sliced Design (FSD)** combined with **Atomic Design** principles:

### FSD Layers
- **App** - Application configuration
- **Pages** - Route components  
- **Widgets** - Complex UI compositions
- **Features** - Business logic
- **Entities** - Domain models
- **Shared** - Reusable components

### Component Organization

Our components are organized using Atomic Design:

- **Atoms** - Basic building blocks (Button, Input)
- **Molecules** - Simple combinations (Card, FormField)
- **Organisms** - Complex sections (Header, Sidebar)
- **Widgets** - Full feature compositions
- **Pages** - Complete application screens

## 🎨 Design System

We use **Material UI v5** as our primary component library with:
- Custom theme configuration
- Dark/light mode support
- Gradient palette extensions
- Tailwind CSS for utilities

## 🚀 Getting Started

Browse the sidebar to explore our components:

1. **Atoms** - Start with basic components
2. **Molecules** - See component combinations
3. **Widgets** - Explore complex UI sections
4. **Pages** - View complete page layouts

## 📝 Component Guidelines

Each component story includes:
- **Interactive controls** - Modify props in real-time
- **Documentation** - Usage examples and API
- **Accessibility** - ARIA labels and keyboard support
- **Responsive design** - Mobile-first approach
